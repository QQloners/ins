# Inscryption 繁中字庫缺字檢查器

這是一個給《邪惡銘刻》模組翻譯用的本機 HTML 工具。

## 功能

- 檢查繁體中文翻譯文字是否包含遊戲繁中包不支援的中文字。
- 以紅字標示不支援字元。
- 列出缺字清單與出現位置。
- 提供部分安全替代字建議。
- 輸入英文名稱時，顯示「典故／諧音辨識」參考，例如北歐神話、歐美民俗神獸、都市傳說、流行文化或英文雙關。

## 使用方式

1. 下載或開啟 `inscryption_zh_checker.html`。
2. 直接用瀏覽器開啟，不需要安裝伺服器。
3. 將翻譯文字貼進左側輸入框。
4. 查看右側檢查結果。

## 注意

- 這不是自動翻譯器。
- `[creature]`、`creature` 這類遊戲程式用字可以保留。
- 「典故／諧音辨識」只提供翻譯前的參考，不會自動決定譯名。
- 工具不包含任何 DLL。
